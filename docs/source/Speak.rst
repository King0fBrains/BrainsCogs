Speak
================

This cog allows you to talk as the bot within a server.

.. note::

        At this moment, this cog does not work in dms, only in servers. This is a planned feature.

Commands
--------
.. rst-class:: command

speak
    Speak as the bot in a server.

    Usage:
        - `[p]speak <message>`

    Example:
        - `[p]speak Hello!`
